# 一个简单的 TODO 项目来练习 ES6 和 Vue
## 安装环境

# 安装 vue-cli 3.x 
npm install -g @vue/cli
```

## 启动项目
``` bash
cd todo_test

# 安装依赖
npm i

# 启动项目 localhost:3000
npm start
```
